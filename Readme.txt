Welcome to this Application, this application is developed in Angular

for using this application you have to:

1. Download this project (you must have installed Angular/Cli and Node)
2. From the terminal, You hate to install "Node Modules" using "npm Install node-modules".
3. Install Bootstrap from NPM using "npm install Bootstrap"
4 execute the server from angular "ng serve -o"
5 In the main page there is a field to search the height in meters and a search button
6 the heights are registered as the JSON format (example 1.96 float datatype)
